Quick Media Application

A Django-based social media aggregator integrating YouTube, Instagram, and Facebook.

Key Features

  • ★ Unified Home Page for easy navigation.
  • ★ YouTube Integration with API-based video fetching and storage.
  • ★ Custom models for Instagram and Facebook posts management.
  • ★ Infinite Scrolling for a seamless user experience.
  • ★ SQL Optimization for efficient data handling.
  • ★ YouTube Recommendation Algorithms for personalized content.
  • ★ AJAX Calls for dynamic content updates and interactions.

Installation and Setup

Follow these steps to get the project up and running:

  1. Clone Repository: 
    git clone [repository-url]
  2. Install Dependencies: 
    pip install -r requirements.txt
  3. Configure Settings: 
    Update settings.py with API keys and database configuration.
    Generate a secret key:
python manage.py shell -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"
  4. Run Migrations: 
    python manage.py makemigrations
python manage.py migrate
  5. Create Admin User: 
    python manage.py createsuperuser
  6. Create Fake Meta Posts & User Data: 
    step 1 - run shell_plus in bash/terminal.
step 2 - open bulk_post.py and copy the code as it is.
step 3 - paste the code in shell as wait to run.
  7. Start Server: 
    python manage.py runserver
